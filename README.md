# aikido-club-website
Planning, design documentation, and custom code for a WordPress.com website built for a local Aikido club

## Overview
A WordPress.com website built for a local Aikido club. This repository documents the planning, design decisions, and custom code used throughout the project.

## Goals
- Improve online presence for the club to attract new members
- Provide clear information for new and existing students
- Showcase club news, class informationm, instructors and club ethos

## Tech Stack
- WordPress.com
- Custom CSS
